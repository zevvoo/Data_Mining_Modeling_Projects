# Microbiome Disease Prediction Shiny App

This Shiny app predicts one of 23 disease classes from an uploaded microbiome TSV file.

## Required Files

The app requires a trained R model file.

```text
microbiome_disease_project/
  webapp/
    app.R
    README.md
  model/
    microbiome_disease_model_R.rds
```

The model file is generated by:

```text
scripts/01_train_predict.R
```

## Required R Packages

```r
install.packages(c("shiny", "data.table", "nnet", "class"))
```

## Run in RStudio

1. Open `webapp/app.R` in RStudio.
2. Click `Run App`.
3. Upload a TSV file with the same structure as `Q2_test.tsv`.

## Run from Terminal

Move to the webapp folder:

Rscript -e "shiny::runApp('.', host='127.0.0.1', port=3838)"

```text
http://127.0.0.1:3838
```

## Upload Format

Upload a TSV file with these columns:

```text
ID
microbiome_1
microbiome_2
...
microbiome_1953
```

After upload, the app shows a prediction preview and provides a CSV download.
